# Dotfiles

This repo contains all my configuration.

## Requirements

- [`direnv`](https://direnv.net/)

## Install

First you must allow direnv environment:

```sh
$ direnv allow
```

Then install dependencies:

```sh
$ pip install --upgrade pip
$ pip install -r dotdrop/requirements.txt
```
